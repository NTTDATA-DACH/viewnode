# Contributing to Ralph Loop

Thanks for your interest in contributing! We welcome pull requests from everyone.

## How to Contribute

1. **Fork** this repository
2. **Create a branch** for your change (`git checkout -b my-change`)
3. **Make your changes** and commit them with clear messages
4. **Push** to your fork and open a **Pull Request** against `master`

## Pull Request Guidelines

- Keep PRs focused — one feature or fix per PR
- Write a clear description of what your change does and why
- Ensure your changes don't break existing functionality
- All PRs require review and approval from a maintainer before merging

## Reporting Issues

Open an issue describing the bug or feature request. Include steps to reproduce for bugs.
